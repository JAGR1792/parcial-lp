\section{Punto 1 y 2}
\subsection*{Objetivo}
Implementar un mini lenguaje CRUD con ANTLR en Python y un visitante de evaluacion en memoria.

\subsection*{Lo realizado}
\begin{itemize}
    \item Se definio la gramatica en \texttt{punto\_1\_\&\&\_2/gramatica/LenguajeNoRelacional.g4}.
    \item Se implemento el analizador en \texttt{punto\_1\_\&\&\_2/analizador.py}.
    \item Se implemento la evaluacion con visitante en \texttt{punto\_1\_\&\&\_2/visitante\_evaluacion.py}.
    \item Se incluyo ejemplo de ejecucion y prueba automatizada.
\end{itemize}

\subsection*{Caracteristicas del lenguaje construido}
\begin{itemize}
    \item \textbf{Enfoque:} lenguaje declarativo de sentencias, orientado a operaciones CRUD sobre colecciones no relacionales.
    \item \textbf{Modelo de datos:} colecciones y documentos tipo objeto (pares clave:valor).
    \item \textbf{Tipos de literal:} cadena, numero, booleano y nulo.
    \item \textbf{Expresiones:} soporte de comparaciones (\texttt{==, !=, <, <=, >, >=}) y operadores logicos (\texttt{Y, O}).
    \item \textbf{Filtrado:} clausula opcional \texttt{DONDE} para \texttt{BUSCAR}, \texttt{ACTUALIZAR} y \texttt{ELIMINAR}.
    \item \textbf{Comandos base:} \texttt{CREAR COLECCION}, \texttt{INSERTAR EN}, \texttt{BUSCAR}, \texttt{ACTUALIZAR FIJAR}, \texttt{ELIMINAR DE}.
    \item \textbf{Ejecucion:} semantica implementada con \textit{Visitor} (EvalVisitor) sobre estructuras en memoria de Python.
\end{itemize}

\subsection*{Ejemplo corto del lenguaje}
\begin{verbatim}
CREAR COLECCION usuarios;
INSERTAR EN usuarios { nombre: "Ana", edad: 20, activo: verdadero };
BUSCAR usuarios DONDE edad >= 18;
ACTUALIZAR usuarios FIJAR activo = falso DONDE nombre == "Ana";
ELIMINAR DE usuarios DONDE edad < 18;
\end{verbatim}

\section{Punto 3}
\subsection*{Objetivo}
Calcular PRIMEROS, SIGUIENTES y PREDICCION para verificar propiedades relacionadas con LL(1).

\subsection*{Lo realizado}
\begin{itemize}
    \item Se reutilizo y adapto el codigo de una tarea anterior para acelerar la solucion del ejercicio pedido.
    \item Se uso como base el repositorio previo: \url{https://github.com/JAGR1792/Tareas_LP}.
    \item Se ejecuto \texttt{python ASD\_Recursivo.py gramatica1.txt} para obtener PRIMEROS, SIGUIENTES y PREDICCION.
    \item Se complemento la justificacion teorica con esta referencia de LL(1): \url{https://www.quora.com/How-can-you-prove-or-explain-with-words-that-an-LL-1-grammar-only-needs-one-lookahead-if-you-make-a-recursive-descent-parser-form-it}
\end{itemize}
